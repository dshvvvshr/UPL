# Resources & References

This document provides curated references to Python tools, libraries, and resources that are particularly relevant to this project and its mission of building ethical AI systems.

For a comprehensive, curated list of Python frameworks, libraries, software, and resources, see:

The Awesome Python repository is an excellent starting point for discovering Python tools across all domains.

## Recommended Python Libraries

Based on this project's focus on AI governance, ethical systems, and API development, here are some particularly relevant categories from Awesome Python:

### AI & Machine Learning

- **[OpenAI Python](https://github.com/openai/openai-python)** - Official OpenAI Python library (already used in this project)
- **[LangChain](https://github.com/langchain-ai/langchain)** - Framework for developing applications powered by language models
- **[Transformers](https://github.com/huggingface/transformers)** - State-of-the-art Machine Learning for PyTorch, TensorFlow, and JAX
- **[LiteLLM](https://github.com/BerriAI/litellm)** - Use any LLM as a drop in replacement for GPT with unified interface

### API Development

- **[FastAPI](https://github.com/tiangolo/fastapi)** - Modern, fast web framework for building APIs (already used in this project)
- **[Uvicorn](https://github.com/encode/uvicorn)** - Lightning-fast ASGI server (already used in this project)
- **[Pydantic](https://github.com/pydantic/pydantic)** - Data validation using Python type hints (already used in this project)
- **[HTTPX](https://github.com/encode/httpx)** - Next generation HTTP client (already used in this project)

### Testing & Quality Assurance

- **[pytest](https://github.com/pytest-dev/pytest)** - The pytest framework (already used in this project)
- **[unittest](https://docs.python.org/3/library/unittest.html)** - Built-in Python unit testing framework (used in this project)
- **[coverage.py](https://github.com/nedbat/coveragepy)** - Code coverage measurement for Python
- **[black](https://github.com/psf/black)** - The uncompromising Python code formatter
- **[flake8](https://github.com/PyCQA/flake8)** - Tool for style guide enforcement
- **[mypy](https://github.com/python/mypy)** - Static type checker for Python

### Configuration Management

- **[python-dotenv](https://github.com/theskumar/python-dotenv)** - Read key-value pairs from .env file
- **[Dynaconf](https://github.com/dynaconf/dynaconf)** - Configuration management for Python

### Async & Concurrency

- **[asyncio](https://docs.python.org/3/library/asyncio.html)** - Built-in asynchronous I/O framework
- **[aiohttp](https://github.com/aio-libs/aiohttp)** - Asynchronous HTTP client/server framework
- **[Celery](https://github.com/celery/celery)** - Distributed task queue

### Security

- **[cryptography](https://github.com/pyca/cryptography)** - Cryptographic recipes and primitives
- **[python-jose](https://github.com/mpdavis/python-jose)** - JavaScript Object Signing and Encryption for Python
- **[Authlib](https://github.com/lepture/authlib)** - Ultimate Python library for OAuth and OpenID Connect

### Development Tools

- **[ipython](https://github.com/ipython/ipython)** - Enhanced interactive Python shell
- **[Poetry](https://github.com/python-poetry/poetry)** - Python dependency management and packaging
- **[pipenv](https://github.com/pypa/pipenv)** - Python development workflow for humans
